# public、private 和 protected

public 为公有变量，在作用域外也可以调用。

private 和 protected 在作用域外都不能调用，基本只能在类内的函数功能实现中使用。private 与 protected 的区别是：

private 变量不能被继承， protected 变量可以被继承

# 继承

继承分为公有继承、保护继承和私有继承，一般只用公有继承

class A : public B, protected C, private D
{

}

保护继承会把公有变量和保护变量继承为保护变量，私有继承会把公有变量和保护变量继承为私有变量。

# 妙用 const、inline 优化运行速度

# const xxx func () 和 xxx func () const 的区别

const xxx func () 表示返回值是常量。

xxx func () const 表示不改变成员变量，这意味着 func () 的实现方式中调用的成员函数都得同样是 xxx func () const。

# inline

inline 同样是函数的修饰词，它用于建议编译器在编译时将函数的定义插入到每个函数调用的地方，这样做的好处是减少函数调用的开销，但只有在函数体较小 （如简单的计算或赋值） 的情况下才适合使用。
