# 从 SolidWorks 入门 Autocad2016 教程

# 写在前面

用惯了 solidworks 的朋友们可能刚接触 autocad 时会很不习惯，因此笔者希望通过这篇博客帮助读者尽快入门 CAD。笔者的 AutoCAD 水平也不高，且是机械专业学生，只会绘制基础的机械装配图，无法给其他专业同学带来较多启发，故仅作参考。

# 基本界面

CAD 的具体分区如上图所示。一般的操作流程都是首先建立各个图层（如粗实线层，细实线层，中心线层，剖面线层，文字层等），将每个图层的线宽等等进行基本的设置，然后选择所需的图层，选用可视化命令 / 输入命令行 -----> 选择合适的捕捉 / 正交关系 ------> 绘图。

要想要快速绘图，掌握上述的快捷键是很有必要的。此外，还有一些必备知识。鼠标单击是框选，长按是曲线选择。单击以后往左选和往右选是完全不同的结果。往右选是选中线条完全处于框选区域内的线条，往左选则是有接触则选中。按住 shift + 框选则是删去选区。

​ 接下来，我们尝试连续绘制几条线段。按下 L+space 后鼠标点击两点绘制出线段，然后按下 space 就会再次执行线段命令，再次选择两点，就画出了第二条线段。重复上述操作，你就熟悉了 CAD 快速重复命令的过程。

​ 熟悉 SW 的朋友们遇到的最大问题主要有三点，一是 CAD 画完的图形无法改变参数，二是图层意识比较淡薄，三是对象捕捉相当地苦恼。下面，我将重点介绍这三个方面。

# 画完的图形无法改变参数

首先，我个人推荐在 CAD 上画图还是先算完再画，而不是先画了再调参数，这是从 SW 到 CAD 画图需要转变的思想。但是如果你真的想要先画完线再调参数，可以采用下面的方法。

len->T->尺寸->选择对象 不过这个指令仅适用于线段，对于圆不适用。

# 图层意识

为了打印出来的 pdf 能够直接显示不同的线条粗细，可以预先设置好图层，这样画完图后就可以直接打印，而不用一根一根线去分图层。

点击图层特性（命令为 layer），打开如下界面。下面是我的图层设置，可以参考一下。

如果想要最终打印出来的 pdf 文件的线条只有黑白色，可以在 ctrl+P 中的打印样式表选择 monochrome 。

# 对象捕捉

对于 sw 来说，对象捕捉基本上不用改，用它的默认设置就能完成绝大部分的三维建模，但是对于 cad 而言，对象捕捉在整个绘制过程是要不断调整的。

# 正篇・画线

  1. 命令：线段 快捷键：l

​ 与之对应的相关指令有 F12，正交 shift，角度 <。

​ 是否启用 F12 会影响你画线的结果。是否启用也会影响到它的提示功能，读者可以视具体情况按 F12。

​ 当画线要画水平或者竖直线时，按住 shift 或者启用右下角的正交开关。

画线时通过输入 a,b 来实现直角坐标，通过输入 a<theta 来控制角度（没有显示指定下一点的模式下要连续按好几次 <)。

  1. 命令：样条曲线 快捷键：spl

在绘制剖面线时，常需要使用样条曲线。spl 命令通常与对象捕捉的最近点相结合。当取完点后，右键选择确认即可。

# 正篇・标注

对于标注问题，学会一个指令几乎就能克服全部困难 ——dimsty (快捷键 dimst)。

・如何标注圆柱的 $\varphi $?

  • dimsty 默认选用的是 GB-3.5，选择 IB-3.5 后置为当前，接下来标注的就都是 $\varphi $ 了，当要标注线段的时候记得切换回来。

・如何改变标注箭头以及文字的大小？

  • dimsty 将当前选用的标注样式修改，在符号和箭头处可以改箭头大小，在文字处可以调整文字大小
