# voron 操作指南

# 写在前面

​ voron 打印机是笔者投入了大量精力复刻的成果，目的是为工作室成员们提供一台工作室专属的高精度 FDM 打印机。由于 voron 打印机硬件维修极为麻烦，基本需要两个人同时在场才较好维修，所以请大家使用时务必好好爱惜。

​ 请注意，本打印机不支持打印 ABS 和 TPU 材料（缺少相关配件），打印时不需要涂固体胶或者粘贴美纹纸，取下打印机时不需要刮刀。在使用 voron 打印机前，请务必仔细阅读操作指南。打印成功与否与第一层紧密关联，请保证第一层正常打印。<font color=red> 不要用手拖拽挤出头位置，这会导致电机与皮带轮松掉 </font>。

# 一、软件说明

# 1. 软件的安装

​ voron 打印机对于切片软件有特殊要求，大家常用的 cura 并不能支持 voron~~(我没试过)~~，要使用 voron 打印机，推荐下载 PrusaSlicer，下面是该软件的下载链接：

​ https://www.prusa3d.com/page/prusaslicer_424/

​ 此外，支持 voron 打印机的软件也包括 ultimaker，但是笔者尝试以后发现 ultimaker 的打印路径规划比较奇怪，这里不作推荐。

​ 后面有关切片软件的介绍皆以 PrusaSlicer 为例。安装过程较为简单，在这里作简要说明。

​ 安装不需要装在 C 盘，可以自己选择位置，如下图，然后点击 next。

出现下图页面，按照默认设置即可，直接点击 next。

​ 安装成功后，桌面上出现 PrusaSlicer 和 Prusa G-code Viewer 两个快捷方式（不出现 G-code viewer 也正常）。

# 2. 软件的配置

​ 第一次打开 PrusaSlicer，它会要求对打印机进行初始的选择与配置。点击左侧其他供应商，选择 Voron 后点击继续。

​ 选择 voron V2 Afterburner 系列的 Voron v2 350mm3 的 0.4mm 喷嘴，如下图所示。350mm3 是指 voron 的尺寸，Afterburner 是近程挤出机，voron 版本为 v2.4。点击继续。

​ 耗材选择如下图所示。

​ 其他的操作均按照默认即可，配置结束。

# 3. 软件的基本使用

​ 第一次进入 PrusaSlicer，它右侧的打印机选项（红色方框框选区域）不会自动切换为 voron，请手动切换。打印设置可能也要自己重新选取。左上角的打印设置、耗材设置、打印机设置等都可以自己调节参数。

​ 切片操作和 Cura 类似，将 STL 导入后，点击右下角的现在切片，切片成功后右下角按钮变为 “导出 G-code”，得到 ".gcode" 文件即可。需要注意的是，如果发现 stl 文件不太对，要重新切片，可以点击左下角方格重新进入 3D 模型阶段。其他功能请自行摸索。

# 二、硬件操作基本介绍

​ voron 打印机启动需要确保三个插头插上，一个是 voron 自带三孔插头，该电线为蜘蛛主板供电；二个是 usb-typeC 线，通过连接 5V2A 以上的手机充电器，为树莓派供电；三是保证小米路由器通电，voron 的有线与路由器连接。满足上述三个条件后，按下开关启动，启动成功后 2min 以内正面下方显示屏亮起且正常显示文字（非纯绿色）。

​ 接线图

​ 正常启动图 (框选区为急停按钮)

​ voron 显示屏下方有一个按钮，为急停开关，紧急情况下按下可终止打印。此时只有重启打印机才能恢复对打印机的控制。

​ voron 后方放置耗材，后方支架上有一个尼龙柱方便耗材滚动，请不要取下。耗材尽量使用易生 esun 的 PLA。之前使用劣质的 PLA 发生了严重的堵喷头现象，处理得很艰难。

# 三、控制台 fluidd

# 1.fluidd 简介

​ 电脑连接网络 kaola 或者 kaola_5g，密码为 12345678。此时可能会跳出小米路由器报错，这是正常现象，忽略即可。(因为确实没网，只有本地的局域网)

​ 连接上网络后，不用在意电脑如下图标。打开浏览器，在网址栏输入 192.168.31.197，回车，进入 fluidd。

​ 黄色警告为正常现象，不用在意，红色警报才影响使用。主页面主要有七个个分页面，Home 和 Jobs 为平时需要用到的页面。Home 页面用于控制打印机，Jobs 用于上传 gcode 等文件。Configuration 可以调节打印机的一些参数，请不要随意改动。下面将简要介绍一下按钮的基本功能。

​ 按钮１控制打印机ｘｙ回零，按钮２控制ｚ轴回零。区域三可以手动输入目标温度，建议打印前可以提早输入，输入温度后按下回车即可迅速升温。推荐挤出机温度为 210°，热床温度为 65°，请注意不要把挤出机温度设置超过 220°，挤出机部分有很多 ABS 打印件，高温会变形。当初就是因为挤出头堵了尝试高温加热疏通导致打印件变形，现在打印件仍有缺陷

​ 此外，由于缺乏配件，请不要用这台打印机打印 abs 材料。

# 2. 打印基本流程

​ 先在 Home 的 thermals 里输入合适的挤出机温度和热床温度并回车，然后把 PrusaSlicer 导出的 gcode 文件上传。上传方式为把文件直接拖拽到 Home 的最下侧的右方的 Jobs 处，或者直接选择 Jobs 这个页面，将文件拖入。正确的文件 fluidd 大概率是能显示缩略图的，且带有 voron 的图标，如下图。

​ 上传成功以后，右键该文件，选择 print 即可。此时打印机会先 xyz 回零，然后会进行四轴调平（具体表现为打印机挤出头分别在打印平台四个角落上下调节，会重复一定次数，视打印机平台水平度情况定）。

​ 打印过程中，最重要的是第一层的打印效果，第一层没问题，后面基本都不会出错，不排除出现粘不住的情况所以打印第一层的时候人尽量在场。事实上，热床和挤出机的加热效率很高，基本上不需要等待多长时间就能够正式开始打印。

​ voron 是可以实时控制的。因此，在打印第一层时，可以手动微调 z 轴高度来实现较好的底层打印效果。按下 print 键后，Home 页面会有所改变。

​ 在 tool 这一栏即可实时调节 z 轴偏移量。

​ 打印暂停或者未开始 print 时，tool 能控制挤出头，换丝完成后可以先 extrude 保证充分换丝。

​ 像图中红色选区就是 z 高度过低的打印效果，是失败品，而外圈这种展开得较充分的走线，则是比较合适的高度。（这是我在打印底层打印了一半时手动减小 z 轴偏移量导致中间区域过低，打印效果不佳。）实际打印时，听声音判断高度是否合适。如果出现挤出头在底板上刻划的声音，则应该把偏移量往上调；如果走线发现太细或者翘起则稍微下调偏移量，使得走线均匀展开。

下面是一些常见的指令

G28 X Y                     //xy回零，效果与按钮相同
G28 Z                       //Z回零，效果与按钮相同
Quad_Gantry_Level           //4轴调平

# voron 常见报错及维修指南（待补充）

  • 问题：不能识别的指令

    方案： 切片的时候 Prusa 的打印机没有切换为 voron

  • 问题：按了 print 后，执行完四轴调平程序就停了，而且报错 out of tolerance（大概，记不清了）

    方案： 可能是四轴调平失败了，四轴调平执行超过五圈仍然没有达到误差范围内则会终止程序。建议直接在代码栏输入四轴调平命令尝试几次，看看能否调平。

    Quad_Gantry_Level

    如果还是不能调平，则应该是皮带过于松的问题，请联系我维修。

  • 问题：打印机失去响应

    方案：

    尝试点击 RESTART KLIPPER 和 FIRMWARE RESTART。如果是因为按下了 voron 的急停键，则重启所有电源。

  • 问题：打印机发出咔哒声

    方案：换耗材的时候耗材没有成功进入挤出机，建议把耗材剪出三角口后再伸入尝试。

  • 问题：机器回零的时候疯狂抖动

    方案：应该是限位开关被线路挡住了导致限位开关无法触发，把电线拨开重新运行即可。
